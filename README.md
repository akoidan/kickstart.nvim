Forked from [kickstart](https://github.com/nvim-lua/kickstart.nvim)

# Hotkeys
- `space f` - format
- `ctrl w s`, `ctrl w v` - split
- `\` - toogle left panel
- `space s g` - search grep
- `space s f` - search files
